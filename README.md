# nginx

[![Build](https://github.com/gsmlg-ci/nginx/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/nginx/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/nginx`
- `ghcr.io/gsmlg-dev/nginx`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/nginx:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/nginx:latest
```

## Build

```bash
docker build -t gsmlg/nginx .
```
